# SignleGoldenRetiver- messaging app that utilizes only 1 Kafka topic for message handling

## Features

- [ ] Sending messages in channels
- [ ] Joining/Leaving channels
- [ ] User sign up and authenthication via naive username/password login
- [ ] Showing Online/Offline status of users
- [ ] Visualizing message flow
- [ ] All that via simple web UI!

### Web UI

- Simple HTML/CSS/Basic JavaScript, nothing too fancy
